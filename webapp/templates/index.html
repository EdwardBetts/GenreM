GenreM - Music Genre Classifier

(Submitted to Sahu Sir)

Project


The project aim was to make a music genre classifier. The project involves use of Machine Learning Techniques of classification, specifically Logistic Regression along with Gradient Descent.

How does it classify Music?


The present music genre classifier can categorize the music into two categories: Metal and Classical. The line of difference between the two genres is quite clear. However, concretely the technique of Logistic Regression along with Gradient Descent is used to classify the music such that the algorithm gives us the probability of the song being close to metal. The code is written in high-level languages like Python and Octave.

Future Scope


The future scope of the classifier is to categorize in many music genres. This would involve using more sophisticated features for music classification.
